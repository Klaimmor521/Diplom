## Hi! Follow these steps to install and run my app:
1. Clone project via the https link: `https://github.com/Klaimmor521/Diplom.git` in your IDE.
2. Navigate to the project folder and activate the virtual environment: `.\.venv\Scripts\Activate.ps1`.
3. Then, run app: `streamlit run app.py`.
4. To stop the app, press `Ctrl+C` in the terminal.
5. To deactivate the virtual environment, run: `deactivate`.

Soon I'll add an `.exe` file, so you could run this app on your PC *easily*!
